当用户使用简单的账户密码登录后，系统通常不会直接得知用户的公钥和私钥。相反，系统会使用用户提供的账户密码来解锁加密的本地存储，其中包含用户的公钥和私钥信息。

具体来说，一般的流程如下：

用户输入账户密码： 用户在登录时输入他们的账户密码。

密码验证： 系统会验证用户提供的密码是否与其本地存储的账户密码匹配。

解锁本地存储： 如果密码验证成功，系统将解锁本地存储，这里包含了用户的加密私钥。

导出公钥和私钥： 一旦本地存储被解锁，系统可以通过相应的加密算法从中导出用户的公钥和私钥。

提供服务： 用户登录成功后，系统可以提供所需的服务，例如查看余额、发送交易等。
